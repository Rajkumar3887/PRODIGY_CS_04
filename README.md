# PRODIGY_CY_04 – Simple Keylogger

## Overview
This project is a basic and ethical keylogger created for educational purposes as part of the Prodigy InfoTech Cyber Security Internship. It records keystrokes on the user's own system and stores them in a text file.

## Features
- Logs key presses in real-time
- Saves logs to `key_log.txt`
- Stops when the user presses ESC
- Built using Python and pynput

## Ethical Note
This tool is strictly for learning and must ONLY be used:
✔ On your own device  
✔ With explicit permission  

Unauthorized use is illegal.

## How to Run

1. Install dependencies:
- pip install pynput


2. Run the script:
- python keylogger.py


3. Type anywhere. Logs will appear in `keylog_results.txt`.

4. Press ESC to stop logging.
